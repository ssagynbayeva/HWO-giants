#list_of_contaminants.txt
#for each object in the HPIC this list gives all potential contaminants from Gaia DR3 within 10"
#note that this is for epoch J2016, and not 2040 when HWO will (hopefully) launch
##columns

obj_name: unique object name in the HPIC
contaminant_Gaia_DR3_id: Gaia DR3 id of the potential contaminant
angDist: angular separation between HPIC object and contaminant (arcsec)
Gmag: Gaia G magnitude of the contaminant
